Tomcat Rails

Tomcat Rails allows you to run a rails application within an embedded Tomcat container.

Apache Tomcat is a lightweight java web server developed and released under the Apache Software Foundation.

This project has just born, there are a lot of things to do. Feel free to add new features, tests, patches…

installing

jgem install calavera-tomcat-rails -s http://gems.github.com

usage

cd myrailsapp
jruby -S tomcat_rails

getting the source

from github: http://github.com/calavera/tomcat-rails/tree/master

git clone git://github.com/calavera/tomcat-rails.git
